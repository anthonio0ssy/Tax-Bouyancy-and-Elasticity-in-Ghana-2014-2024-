# Tax Analysis Using Log-Log Regression in R

## Overview
This project applies econometric techniques in R to analyze tax data. The focus is on understanding the relationship between key variables using a log-log regression model, which allows for the estimation of elasticities.

## Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Apply log-log regression modeling
- Interpret coefficients as elasticities
- Visualize relationships between variables

## Tools and Technologies
- R
- ggplot2
- dplyr
- lmtest
- readxl
- sandwich
